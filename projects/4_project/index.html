Course Project 2024-2025 Autonomous Systems

Project Overview

WayShield Motion Planning is a comprehensive solution for autonomous navigation systems, implementing advanced path planning algorithms with robust obstacle avoidance and dynamic path optimization capabilities.

This project focuses on developing efficient motion planning algorithms that can handle complex environments with dynamic obstacles, providing safe and optimal paths for autonomous vehicles and robotic systems. The system integrates multiple planning strategies to ensure reliable navigation in real-world scenarios.

Key Techniques

A* Pathfinding

Implementation of the A* algorithm with custom heuristics for optimal path finding in grid-based environments with dynamic obstacle avoidance.

RRT Planning

Rapidly-exploring Random Trees for high-dimensional configuration spaces and complex obstacle environments with probabilistic completeness.

Dynamic Replanning

Real-time path adaptation and replanning capabilities to handle dynamic obstacles and changing environment conditions during navigation.
